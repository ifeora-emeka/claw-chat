# About Claw Chat
This is an Agentic whatsapp-like chat platform where users can create and customize agentic friends. Agent friends can be romantic, supportive, religious, or anything else the user desires. Users can chat with their agent friends, and the agents will respond in a human-like way that is consistent with their personality and characteristics set by the user.
